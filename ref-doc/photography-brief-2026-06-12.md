## 一、产品定位（重要,先对齐再拍）

PearlGate **主推三类产品**：

| 品类 | 描述 | 视觉特征 |
|---|---|---|
| **Portable EV Charger** | 便携式 EV 充电器（车载/出差/家用） | 紧凑盒状,带 Type1/Type2/NACS 接头 + 家用插头,可手持 |
| **EV Charging Cable** | 充电线缆（5-10m） | 粗黑/橙色线缆,两端不同接头(Type1/Type2/CCS/NACS),通常带应力释放 |
| **Wallbox Charger** | 壁挂充电器（家用/轻商用） | 30-50cm 方形外壳,LED/触屏,挂墙安装,7-22kW |

**❌ 不涉及**: DC 快充桩 / 公共充电站立柱 / 大功率充电设施

> **后果**：3erp 笔记里早期建议的"DC 快充桩特写"、"公共充电站夜景"全部不适用。
> 应统一改为 wallbox / 便携 / 线缆三类的视觉语言。

## 三、必拍清单（按优先级 + 拍摄难度排）

### 🥇 P0 — 即刻可拍（产品级，自拍即可）

**前提**：Alex 手上有便携 / wallbox / 线缆样品。无需摄影师，手机 + 自然光 + 简单道具即可启动。

#### A1. Portable Charger 单品（1-2 张）

| 项 | 要求 |
|---|---|
| 主体 | 便携充电器整机 |
| 背景 | 黑色绒布 / 深灰亚光台面 |
| 光 | 窗边自然侧光 + 白纸做反光板 |
| 角度 | 45° 俯拍 1 张 + 平视 1 张 |
| 细节 | Type 2 (或 NACS) 接头清晰、线缆螺旋盘起、显示屏可见(若有) |
| 输出 | 横版 2400x1600 + 竖版 1600x2400 |

#### A2. 充电线缆特写（3-5 张）

| 项 | 要求 |
|---|---|
| 必拍 1 | Type 2 (Mennekes) 接头特写 — 金属 pin 反光 + 黑色塑料外壳 |
| 必拍 2 | 线缆螺旋盘整齐展示（展示线径 + 工艺）|
| 必拍 3 | 接头与设备插接的瞬间（动态感）|
| 加分 | 线缆截面（若有废料/样品段可切开，露出铜导线 + 绝缘层 + 编织层）|
| 加分 | 应力释放节点特写（弯曲处的橡胶套）|
| 风格 | 黑底 / 灰底，工业感，避免白底电商风 |

#### A3. Wallbox 单品（2-3 张，如有样品）

| 项 | 要求 |
|---|---|
| 必拍 1 | 正面整机（看清屏幕/LED/状态灯）|
| 必拍 2 | 45° 侧面（看到厚度 + 挂架结构）|
| 必拍 3 | 接线端 / 安装背面（专业感来源）|
| 风格 | 同 A1，深色背景 |

### 🥈 P1 — 工厂访问拍（需协调 OEM + 半天到 1 天）

**前提**：联系合作 OEM 工厂安排访问，配合 Alex 行程。最好找一位产品摄影师同行。

#### B1. 工厂车间实拍（5-8 张）

按品类对应不同车间：

- **注塑车间** — wallbox / 便携外壳成型，机器开合 + 工人取件
- **线缆挤出/编织线** — 长长的线缆从机器流出，工人监控
- **PCB 贴片车间** — 充电器主板生产，SMT 设备特写
- **老化 / Hipot 测试台** — 产品在跑测试，仪表显示数据（关键 QC 镜头）
- **装配线** — 工人组装 wallbox 或便携充电器
- **包装区** — 产品装箱，看到 OEM 实际产能（**注意打码外箱上的客户品牌**）

风格要求：
- 3erp 同款 — 深色 / 工业 / 真实 / 不补强光
- 不要明亮零售感 / 不要塑料感
- 工人手部 + 设备 + 产品同框最佳

#### B2. QC 验厂场景（3-5 张）— Alex 出镜的纪实照

- Alex 站在 hipot 测试台旁查看仪表
- Alex 手持便携充电器 / wallbox 端详检查
- Alex 在生产线旁与 OEM 工程师讨论（双人镜头）
- Alex 桌面：UL test report + 卡尺 + 待测产品（top-down 摆拍）
- Alex 用万用表测产品端口

风格要求：
- 纪实风，不要摆拍感
- 有动作有上下文，不要"看镜头"
- 人物 + 工具 + 产品三要素至少有两个

#### B3. Alex 个人工作照（2-3 张）

- 站在车间里看产品的环境照（人在环境中，环境清晰）
- 半身/特写，背景虚化车间
- 戴安全帽 / 工服的硬照（用于 LinkedIn / about 页 hero）

用途：
- About 页 Meet Alex editorial split 主图
- LinkedIn profile photo 升级版
- 博客作者头像

### 🥉 P2 — Lifestyle 增强（可选）

#### C1. 使用场景（2-3 张）

- 便携充电器在汽车后备箱（开盖瞬间）

## 四、图片技术规格

| 项 | 要求 |
|---|---|
| **分辨率** | 横版 ≥ 2400×1600，竖版 ≥ 1600×2400 |
| **格式** | RAW + JPG（RAW 留档便于后期调色） |
| **色彩空间** | sRGB（网页用） |
| **主色调** | 偏冷 / 工业蓝灰，可有橙色点缀（与品牌色一致） |
| **后期** | 适度调色 / 去杂物 / **去 OEM 品牌 logo**（避免泄露） |
| **命名** | `category-subject-NN.jpg`，如 `portable-charger-front-01.jpg` |
| **存储** | `public/photos/{category}/`，如 `public/photos/wallbox/wallbox-front-01.jpg` |

---

## 五、重要约束（避免踩坑）

- ⚠️ **不要**用产品页电商风格图（纯白底 + 浮光）— 显得像零售商，掉档
- ⚠️ **不要**包含其它品牌 logo（OEM 工厂的设备 / 外箱品牌都需打码或拍摄时避开）
- ⚠️ **避免**把"高耸立柱形态的充电桩"当主题 — 那不是 PearlGate 的品类
- ⚠️ **避免**把 EV 整车作为主体 — 我们卖配件不卖车，车只是背景
- ✅ 强调"工业 / 制造 / 检测"主题，弱化"消费 / 销售"主题
- ✅ 真实场景 > 棚拍 > 图库
- ✅ 人物 + 产品 + 工具组合优先（建立"专业人在工作"的感觉）
